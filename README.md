Swedish_data - Performing Exploratory Data Analysis(EDA)
   This project performs Exploratory Data on a Swedish Insurance company.The goal is to understand the data, clean, visualize and extract emaningful informatio using R Programming.

PROJECT FEATURES
Data loading
Checking for missing values
Summary Satistics(Mean, Median, etc.)
Data Cleaning
Visualizations with ggplot2
Correlation Analysis
Pattern Revealations

TOOLS & LIBRARIES USED
gggplot2
read.csv
R

HOW TO RUN THE PROJECT
Download or clone this repository
Open Rstudio
GO to file and choose the project folder
Open the main script(Swedish_data.Rmd)
Click the run button in RStudio
